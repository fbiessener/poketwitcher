About:

PokéTwitcher is Fiona Biessener's final project for Hackbright Academy. It is developed as a full-stack application with a PostGreSQL database, Python/Flask backend and Bootstrap/HTML/Javascript(AJAX, JSON) frontend. APIs utilized: PoGo API (include link) for the Pokemon Go data, PokéAPI's (link github) github sprites resource, Cloudinary for image hosting. Chart.js was used to generate the pie charts.
